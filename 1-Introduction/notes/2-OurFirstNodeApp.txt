===================
Our first Node app
===================

Okay so we just created a new 
directory, first-node-app, then 
created a new js file called 
first-app.js 

Then added a console log 'hello from 
node js' 

In the terminal, in the current directory 
type 'node first-app.js' and it logs 
hello from node.js to the console 

==============================
Writing to a new file 
==============================

Next we demonstrate how to write 
a new file with node, first we 
require the 

const fs = require('fs')


fs.writeFileSync('hello.txt',
 'hello from nodejs fs writefilesync')

 This writeFileSync will take two args, 
 first is the file name, second is the content 
 to write inside the file
